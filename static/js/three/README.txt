Place the following files in this folder (download from https://unpkg.com/three@0.152.2/build/ and /examples/jsm/):

- three.module.js
- OrbitControls.js (from examples/jsm/controls/)
- GLTFLoader.js (from examples/jsm/loaders/)

All files must be ES modules. This folder is for local Three.js modules for browser import.
